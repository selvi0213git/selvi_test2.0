= Korea SNS =

* You can Insert share buttons for korean in contents post or page.
* The post can be inserted into the button information, information before, after, or during you.
* You can set the plug-in settings page before, is inserted after the selection.
* If you enter a short code in the article is my content is inserted as follows: [korea_sns_button]
* Share Buttons - facebook, twitter, google, kakaotalk, kakaostory, naver line, naver band, naver blog.
* More Information : http://icansoft.com/product/korea-sns/
* Support Forum : http://facebook.com/groups/koreasns

== Installation ==

= 설치하기 =

* “워드프레스 데쉬보드 -> 플러그인 메뉴 -> 플러그인 추가”로 들어갑니다.
* “korea sns”로 검색하여 “Korea SNS”를 설치합니다.
* 플러그인 메뉴에서 korea SNS “활성화” 버튼을 클릭 합니다.
* 설정 화면에 Kakao API Key를 입력합니다.
* ! 본 페이지에서 직접 다운로드 받을 수도 있습니다. 이 경우, FTP를 통해 /wp-content/plugins/ 디렉토리에 업로드하세요.

= Installation =

* Find Korea SNS plugin and install plugin in wordpress dashboard.
* Activate the plugin through the Plugins menu in WordPress
* Select position option in Dashboard > Setting > Korea SNS
* Insert Kakaotalk API Key
* ! You can download from this page. In this case, upload the korea_sns directory into the /wp-content/plugins/ directory by FTP.
